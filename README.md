# Сайт-визитка Федорцова Александра

Мобильный-first сайт-визитка с интерактивными контактами, формой записи студентов на практику, блоком маркетинговых новостей и серверной админкой.

## Запуск

```bash
npm install
npm run dev
```

Сайт: `http://127.0.0.1:5176`

Админка: `http://127.0.0.1:5176/#admin`

## Пароль админки

Админка закрыта паролем. Для локального запуска создайте файл `.env` по примеру `.env.example`:

```bash
ADMIN_PASSWORD=ваш-пароль
ADMIN_TOKEN_SECRET=длинная-секретная-строка
```

Пример лежит в `.env.example`. Если переменная `ADMIN_PASSWORD` не задана, локально используется временный пароль `admin12345`. Перед размещением на GitHub/хостинге обязательно задайте свой пароль через Secrets или переменные окружения сервиса.

## Где заменить фото

Основное фото находится в `public/images/profile-placeholder.svg`.

Чтобы поставить реальное фото:

1. Сохраните файл в `public/images/profile.jpg`.
2. В `src/App.tsx` замените строку:

```tsx
<img src="/images/profile-placeholder.svg" alt="Александр Федорцов" />
```

на:

```tsx
<img src="/images/profile.jpg" alt="Александр Федорцов" />
```

Лучше использовать вертикальное фото 1200x1600 или близкое к этому соотношение.

## Данные

Новости и заявки студентов хранятся в `server/data/db.json`.

Админка позволяет:

- добавлять, редактировать и удалять новости;
- смотреть, редактировать и удалять заявки;
- менять статус заявки: без статуса, на рассмотрении, ожидаю обратную связь, согласовано, отказано.
